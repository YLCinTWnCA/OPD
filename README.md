# OPD 診所排班系統

單一檔案的網頁排班工具，免安裝、用瀏覽器打開即可使用。

**線上使用：** https://ylcintwnca.github.io/OPD/

## 功能
- 班別：週一～五 早/午/晚班、週六 早/午班，每班 4 小時
- 位置：門診 1～3 診（醫師＋護理師）、治療室（護理師）
- 流程：手動放醫師（決定開診）→ 一鍵「自動排護理師」→ 手動微調
- 護理師規則：一天最多兩班（早+午／午+晚／早+晚皆可、可換診間或治療室）、可設排休日、晚班隔天不排早班、不可同診配對（如 A✕a）、4 週時數上限（預設 160 小時）
- 即時違規標紅、時數統計、CSV 匯出、列印／存 PDF、JSON 備份匯出入

## 資料儲存（重要）
資料只存在「**你這台電腦的這個瀏覽器**」（localStorage），不會上傳到 GitHub、也不會跨裝置自動同步。
- 換電腦或備份：用「**資料備份**」分頁 →「匯出備份檔(.json)」，到另一台「匯入備份檔」。
- 每個人、每台電腦、每個瀏覽器的排班資料各自獨立。

## 本機使用
直接用瀏覽器打開 `index.html`（或 `診所排班系統.html`）即可，不需要網路。
